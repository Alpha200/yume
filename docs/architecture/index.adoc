== Quick Overview

*Yume* (夢 – "dream") is a personal AI assistant that fuses several information sources into a single conversational interface:

* *Chat Interface*: Matrix protocol (Synapse homeserver)
* *AI Backbone*: langchain4j-powered OpenAI-compatible LLM agents
* *Proactive Scheduling*: AI-driven self-scheduling for daily summaries and reminders
* *Location Awareness*: Home Assistant geofence triggers
* *Data Sources*: Calendar (CalDAV), public transport (EFA), shopping (KitchenOwl), sports (Strava), health (Garmin Connect), weather (OpenWeatherMap)
* *Memory*: Hybrid MongoDB (documents) + PostgreSQL/pgvector (semantic embeddings)
* *Frontend*: Vue.js 3 SPA with Tailwind CSS / DaisyUI
* *Backend*: Spring Boot 3.5 on Java 21 / Kotlin 2.2
